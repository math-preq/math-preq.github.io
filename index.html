Math-Preq

What?

This project aims to gather all mathematical prerequisities for different STEM topics.

Why?

Many people around the globe need or prefer to self-study mathematics, or even STEM topics; in this case, it could be hard for them to pinpoint what exactly do they need to learn, what is really not necessary to learn, in what order should they study the topics, and where to find good resources for them. This project aims to help these people in the sacred journey of learning.

How?

We are currently preparing a series of roadmaps for every mathematical or math-based subject that our team knows enough about. Below, you may find a list of their links.

Yellow boxes are application boxes (e.g., "The Backpropagation Algorithm"). Blue boxes are "theoretical" mathematical topics (e.g., "A vector space is an abelian group under vector..."; this definition requires knowing some basic Group Theory). Pink boxes are "applicational" versions of the same mathematical topics (e.g., "A vector space is a set of things called vectors which can be added together..."; this definition does not require knowing anything about groups).
